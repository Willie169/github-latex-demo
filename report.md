Subject

GitHub Android app does not render LaTeX math in Markdown files

---

Last known working date



---

What issue are you facing?

The GitHub Android app does not render LaTeX math expressions of all four syntaxes specified in [Writing mathematical expressions](https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/writing-mathematical-expressions) in Markdown files. Instead, they are treated as raw texts.

The same files renders correctly on github.com in a web browser. I don't know whether iOS app renders it since I don't have an iOS device to test.

---

What are the steps to reproduce this?

1. Create a repository and add a Markdown file. Both README.md or not are ok.
2. Add any LaTeX math expression in it.
3. Open the repository on github.com in a web browser.
4. Open the same repository in the GitHub Android app.
